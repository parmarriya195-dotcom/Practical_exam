# 📚 Bookstore Inventory and Analytics System

## 📌 Project Description

This project is a Python-based **Bookstore Inventory and Analytics System** designed to manage bookstore inventory, track sales, and analyze data using visualizations.

---

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📂 Dataset Details

### 1. inventory.csv

* title
* author
* genre
* price
* quantity

### 2. sales.csv

* date (YYYY-MM-DD)
* title
* quantity_sold
* total_revenue

---

## ⚠️ Important Note (File Path)

* In this project, **CSV file location (full path) must be specified** in the code.
* Example:

```python
path_inv = "C:/python-project/practical_exams/inventory.csv"
path_sales = "C:/python-project/practical_exams/sales.csv"
```

* If correct file path is not provided, the program may show an error.
* Ensure the CSV files exist at the specified location before running the program.

---

## ▶️ How to Run

1. Install required libraries:

```
pip install pandas numpy matplotlib seaborn
```

2. Update file paths in the code (important)

3. Run the program:

```
python main.py
```
